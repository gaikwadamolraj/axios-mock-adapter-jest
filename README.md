# axios-mock-adapter-jest
This will showcase the mocking the axios async await call for unit testing.

## Local setup and run
   - ### To run this app, you'll need below node version
     ```sh
      v16.14.0
     ```

   - ### Inastall dependencies
     ```sh
     npm install
     ```
   - ### Run jest tests
     ```sh
     npm test
     ```
